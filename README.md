**MicroService** is a modular e-commerce backend built using **Spring Boot microservices architecture**.  
It consists of independent services for **Product**, **Order**, and **Payment**, along with supporting infrastructure like service registry and config server.

---

## üöÄ Project Overview

This project demonstrates a scalable and maintainable **microservices-based e-commerce backend** using Spring Boot and related technologies.  
Each service encapsulates a core business domain and can be developed & deployed independently. 

### üß± Services Included

| Service | Purpose |
|---------|---------|
| `ProductService` | Manages product catalog & inventory. |
| `OrderService` | Handles order creation & status. |
| `PaymentService` | Processes payments and settlements.  |
| `service-registry` | Registers all microservices (Eureka). |
| `ConfigServer` | Centralized configuration management.  |
| `CloudGateway` | API Gateway to route requests .  |
